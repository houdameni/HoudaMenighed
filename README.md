## Installing

Download the latest extract the zip, and run the exe.

#### Or run it locally

```sh
$ git clone https://github.com/houdameni/HoudaMenighed.git
```
```sh
$ cd HoudaMenighed
```
```sh
$ pip install -r requirements.txt
```
```sh
$ python3 Menighed_Houda.py
```

## Controls

| Key   | Function       |
| ----- | :------------: |
| W     | Move Forward   |
| S     | Move Backward  |
| A     | Move Left      |
| D     | Move Right     |
| Space | Jump         |
